We can change the understanding and forgiving behaviour of JavaScript to some extent using strict mode.
You can switch on the strict mode with the following command in your code. This needs to be the first command in your code. 

    'use strict';

Here is something that works when we don't use strict mode. 

    function sayHi() {
        greeting = 'Hello';
        console.log(greeting);
    }
    sayHi(); // CHECK CONSOLE FOR OUTPUT

We didn't declare 'greeting', so JavaScript did it for us by adding 'greeting' to the top level and it will log 'Hello'.
If we enable 'strict mode', it will have an error. 

    'use strict';
    function sayHi() {
        greeting = 'Hello';
        console.log(greeting);
    }
    sayHi();
